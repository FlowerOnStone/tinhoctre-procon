# tinhoctre-procon

tinhoctre procon

## Getting Started

### Install dependencies

```bash
npm install
cp .env.example .env
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build and start

```bash
npm run build
```

```bash
npm run start:public
```
