# Procon BE

## Create virtual environment

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirement.txt
```

### Linux / MacOS

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirement.txt
```

## Run server

```
python3 manage.py runserver
```
